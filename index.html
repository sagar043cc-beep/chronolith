How to Set Up Gmail SMTP Integration

          

          
          

            
To deliver emails securely through Nodemailer, you must configure a Google App Password:

            
    
              
  1. Enable 2-Step Verification on your Google Account settings page.
    2. 
              
  2. Go to Google Account → Security → App Passwords.
    3. 
              
  3. Generate a custom password, selecting "Mail" and your device type.
    4. 
              
  4. Add the generated 16-character credential to your project's local .env file:
                
    GMAIL_USER=your.address@gmail.com
GMAIL_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx
    
              
    5. 
              
  5. Deploying online? Save these exact variables in your Vercel Dashboard → Project Settings → Environment Variables.
